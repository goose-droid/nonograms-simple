Nonograms







How to play

Click a box once to turn it dark blue.
Click it twice to turn it to orange.
Click it thrice to return it to light blue.

Create a picture in dark blue.

The numbers indicate the number of dark blue boxes there should be in each row and column, split into groups. For example, if the top row has a hint of "2, 3, 3", that means that the top row contains 8 dark blue boxes, split into groups of 2, 3, and 3. There is at least one light blue box between each group of dark blue boxes.

If you want, you can use the orange color to keep track of boxes you know cannot be dark blue. When checking your work, the computer ignores the orange boxes, counting them the same as light blue.

When you are done, check to see if you solved the puzzle using the "Check" button. You may then start another game by using the "Reset" button.

There are eight separate puzzles, and one of these is selected randomly when you click "Reset" or refresh the page.
